# Nmap Network Scanning Project

## Overview
This project demonstrates how I used Nmap to scan networks and identify open ports and services.

## Objective
The goal was to understand how attackers and security professionals use scanning techniques to gather information about a network.

## Tools and Technologies
- Nmap
- Kali Linux
- Windows 11 VM

## Process
- Performed basic scans to identify active hosts
- Used SYN scan to detect open ports
- Analyzed results to determine running services
- Compared different scan types

## Scan Types Used
- SYN Scan
- Connect Scan
- ACK Scan

## Key Skills Demonstrated
- Network scanning
- Port analysis
- Understanding TCP/IP behavior
- Interpreting scan results
